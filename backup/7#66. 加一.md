# 66. 加一

[66. 加一](https://leetcode.cn/problems/plus-one/)
**思路：**
小学加法计算过程模拟
```
class Solution {
public:
    vector<int> plusOne(vector<int>& digits) {
        vector<int> ans(digits);
        int in = 1, n = ans.size();
        for(int i = n - 1; i >= 0 && in > 0; --i){
            ans[i] += in;
            in = ans[i] / 10;
            ans[i] %= 10;
        }
        if(in > 0){
            ans.insert(ans.begin(), 1);
        }
        return ans;
    }
};
```
